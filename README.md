## Deployment

To deploy to github pages, checkout the branch you want to deploy and then run
```
npm run deploy
```

To build a static copy instead, run:
```
npm run build
```

## Development


to install dependencies, cd into folder and run
```
npm install
```

to run a development server:
```
npm start
```
